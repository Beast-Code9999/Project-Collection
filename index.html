My Projects
Booking page

Booking page is a project mostly made with CSS, it is a beautful project designed for hotel display. This website is a great exercise for flexbox design and usage, hence the reason of using flexbox as the main CSS tool.

landing-page-v2

Landin page version 2 is a website with complex design designed by Jonas Schmedtmann, the technologies used for this website are mainly SASS and HTML, there were very little JavaScript involved, but there were many powerful CSS animations I was able to use.

Admin Dashboard

Admin Dashboard uses intermediate level of CSS, It is not the most responsive, scalable, nor complex website, but it is a rather beginner friendly website to make in order to use grid technology from CSS.

Sign-up Form

Frontend sign-up page without any backend, it was a rather fun project for anyone interested in learning about form element and form controls, which is one of the most important element in a website for a business.

Random Password Generator

Random password generator is a project specifically made to streamline the process of generating a password with some funky Fallout theme.

Etch-a-Sketch

Etch-a-sketch was my favourite project to make as it was the most visually interactive and is something that's actually fun to play with when taken seriously.

Calculator

Making a calculator with interactive user interface wasn't the easiest task, it took a lot of revision of code logic and strcuture, this project was rather special because it is one of those projects that can be either very easy or hard to make, depending on the appraoch and logic.

Rock Paper Scissors

Rock Paper Scissors is a highgly recommended project to make as a beginner developer due to its very simple rule. Nevertheless, the project have the potential to get rather complicated depending on how much sassy-ness one wants it to be

Landing-page-v1

A very classic website layout exercise without any JavaScript and mostly CSS to build, this project is recommended for beginners as it introduces one of the most basic layout used on the internet.
